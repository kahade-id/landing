Landing page Kahade dibangun dengan Next.js App Router, TypeScript, dan Tailwind CSS.

## Isi project
- Homepage landing dengan beberapa section pemasaran
- Halaman pendukung: about, contact, privacy, terms, cookies, status, docs, blog, careers, partners, support
- SEO dasar: metadata, robots, sitemap
- Asset dasar: favicon dan og-image

## Menjalankan project
```bash
npm install
npm run dev
```

## Build production
```bash
npm run build
npm run start
```

## Catatan implementasi
- Sebagian section masih memakai inline style karena source awal memang sangat custom.
- Routing internal sudah diarahkan ke pola Next.js dan halaman pendukung nyata.
- Klaim trust/compliance telah dinormalkan agar tidak misleading tanpa bukti pendukung.
